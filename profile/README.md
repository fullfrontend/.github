## What this is

Full Front-End is a personal organization.

Not an agency.  
Not a startup.

It’s a space to design, test and refine:

- automation systems
- self-hosted tools
- simple architectures
- real-world workflows

Everything here follows the same rule:

> If it becomes complex, it’s broken.
